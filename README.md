## :computer: Installation

install this repository: 

```
Git clone https://github.com/dennishva/blok-tech
```

To install Node modules:

```
npm install
```

To run the application:

```
npm run start
```

To connect to the database:

```
DB_URL="" 
```

To deploy with port:

```
PORT=1337
```
